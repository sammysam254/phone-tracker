## 🆕 New Feature: Re-Pair Device

### Location
Main screen → Scroll down → "🔄 Re-Pair Device" button (red)

### What It Does
1. Clears all local pairing data
2. Clears all server data (activities, pairing records)
3. Stops monitoring services
4. Opens QR scanner for fresh pairing

### When to Use
- Switching to different parent account
- Troubleshooting pairing issues
- Testing with multiple accounts

### How to Use
1. Open child app
2. Tap "🔄 Re-Pair Device"
3. Confirm dialog
4. Wait 2-5 seconds
5. Scan new QR code
6. Done!
