📱 Installation & Permission Guide

Complete step-by-step instructions for setting up Parental Control Monitor

Child App Installation

📌 Before You Start:
  • Download the Child App APK from the dashboard
  • Ensure the child device has Android 7.0 or higher
  • Have the parent device ready for QR code pairing
1
Download the APK

Go to the download page and tap "Download Child App"

The APK file will be saved to your Downloads folder

2
Enable Unknown Sources

When you try to install, Android will ask for permission:

  1. Tap "Settings" when prompted
  2. Enable "Allow from this source"
  3. Go back to the installer
3
Install the App

Tap "Install" and wait for installation to complete

Tap "Open" when installation finishes

4
Grant Consent

On first launch, you'll see a consent screen:

  1. Read the consent information
  2. Tap "I Understand and Agree"
  3. Tap "Grant Consent"
5
Scan QR Code

The app will open the QR scanner:

  1. Open the Parent App on parent device
  2. Generate a QR code
  3. Point the child device camera at the QR code
  4. Wait for successful pairing
6
Grant Basic Permissions

The app will request essential permissions:

  1. Location - Tap "Allow all the time"
  2. Contacts - Tap "Allow"
  3. Phone - Tap "Allow"
  4. SMS - Tap "Allow"
  5. Storage - Tap "Allow"
  6. Camera - Tap "Allow"
  7. Microphone - Tap "Allow"
7
Enable Accessibility Service

This is the most important step for screen monitoring!

  1. Tap "Grant Accessibility"
  2. Tap "Installed apps" or "Downloaded apps"
  3. Find and tap "Parental Control"
  4. Toggle the switch to ON
  5. Tap "OK" or "Allow" on the confirmation dialog
  6. Tap "Open Settings" if prompted
  7. Go to Apps → Parental Control
  8. Tap the three dots (⋮) at the top right
  9. Select "Allow restricted settings"
⚠️ Important: Without accessibility service, screen interactions and keyboard inputs won't be monitored!
8
Enable Notification Access

To monitor notifications:

  1. Tap "Grant Notification Access"
  2. Find "Parental Control" in the list
  3. Toggle it ON
  4. Tap "Allow"
9
Disable Battery Optimization

To ensure continuous monitoring:

  1. Go to Settings → Apps → Parental Control
  2. Tap "Battery"
  3. Select "Unrestricted" or "Don't optimize"
✅ Setup Complete! The child app is now monitoring and sending data to your parent dashboard.

Parent App Installation

📌 Before You Start:
  • Download the Parent App APK from the dashboard
  • Create an account on the web dashboard first
  • Have your login credentials ready
1
Download & Install

Download the Parent App APK and install it (same process as Child App)

2
Open the App

Launch the Parent App from your app drawer

3
Login

Enter your dashboard credentials:

  1. Enter your email
  2. Enter your password
  3. Tap "Login"
4
Generate QR Code

To pair with a child device:

  1. Tap "Add Device" or "Generate QR Code"
  2. A QR code will be displayed
  3. Keep this screen open
  4. Use the child device to scan this code
5
View Dashboard

After pairing, you can:

  • View all monitoring data
  • Switch between multiple devices
  • Access all features from the web dashboard
  • Receive real-time updates
✅ Parent App Ready! You can now monitor your child's device from your phone.

Detailed Permissions Guide

⚠️ Critical: All permissions must be granted for full functionality. Missing permissions will result in incomplete monitoring.
📍
Location Permission

Why needed: Track device location in real-time

How to grant:

  1. When prompted, tap "Allow all the time"
  2. Or: Settings → Apps → Parental Control → Permissions → Location → Allow all the time
👥
Contacts Permission

Why needed: Display contact names in call logs and messages

How to grant: Tap "Allow" when prompted

📞
Phone Permission

Why needed: Monitor incoming and outgoing calls

How to grant: Tap "Allow" when prompted

💬
SMS Permission

Why needed: Monitor text messages

How to grant: Tap "Allow" when prompted

📁
Storage Permission

Why needed: Access photos and save monitoring data

How to grant: Tap "Allow" when prompted

📷
Camera Permission

Why needed: Remote camera access and QR scanning

How to grant: Tap "Allow" when prompted

🎤
Microphone Permission

Why needed: Record calls and remote audio

How to grant: Tap "Allow" when prompted

♿
Accessibility Service (MOST IMPORTANT)

Why needed: Monitor screen interactions, keyboard inputs, and app usage

Detailed Steps:

  1. Click "Grant" button in the app
  2. Click "Installed apps" or "Downloaded apps" in the list
  3. Click "Parental Control" from the app list
  4. Click "OK" or toggle the switch to ON
  5. Open Settings (if prompted)
  6. Go to Apps → All apps
  7. Open "Parental Control"
  8. Select the three dots (⋮) at the top right corner
  9. Allow restricted settings - This is crucial!
⚠️ Without "Allow restricted settings":
  • Screen interactions won't be captured
  • Keyboard inputs won't be monitored
  • App usage tracking will be limited
🔔
Notification Access

Why needed: Monitor all notifications

How to grant:

  1. Settings → Apps → Special app access → Notification access
  2. Find "Parental Control"
  3. Toggle ON
🔋
Battery Optimization

Why needed: Prevent Android from stopping the monitoring service

How to disable:

  1. Settings → Apps → Parental Control
  2. Tap "Battery"
  3. Select "Unrestricted" or "Don't optimize"

Troubleshooting Common Issues

❌
App Won't Install

Solutions:

  • Enable "Install unknown apps" for your browser/file manager
  • Check if you have enough storage space
  • Try downloading the APK again
  • Restart your device and try again
❌
QR Code Won't Scan

Solutions:

  • Grant camera permission
  • Ensure good lighting
  • Hold the device steady
  • Try increasing screen brightness on parent device
  • Make sure QR code is fully visible on screen
❌
No Data Showing in Dashboard

Solutions:

  • Check internet connection on child device
  • Verify all permissions are granted
  • Ensure accessibility service is enabled
  • Check if battery optimization is disabled
  • Restart the child app
  • Wait a few minutes for data to sync
❌
Keyboard Inputs Not Showing

Solutions:

  • Verify accessibility service is enabled
  • Check "Allow restricted settings" is enabled
  • Go to Settings → Apps → Parental Control → Three dots → Allow restricted settings
  • Restart the device
❌
Location Not Updating

Solutions:

  • Ensure location permission is set to "Allow all the time"
  • Enable GPS/Location services on the device
  • Check if device has internet connection
  • Disable battery optimization for the app
❌
App Keeps Stopping

Solutions:

  • Disable battery optimization
  • Add app to "Protected apps" or "Auto-start" list
  • Check if device has enough RAM
  • Clear app cache: Settings → Apps → Parental Control → Storage → Clear cache
