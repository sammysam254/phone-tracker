📋 Installation Instructions

🎯 Which App Should I Download?

Child App: Install on your child's device for monitoring their activities
Parent App: Install on your phone to access the dashboard and control features

1

Download Both Apps

Download the Child App for your child's device and Parent App for your phone.

2

Enable Unknown Sources

On both devices: Settings → Security → Install unknown apps and enable for your browser.

3

Install Child App First

Install the Child App on your child's device and complete the setup wizard.

4

Grant All Permissions

Allow all requested permissions on the child's device for full monitoring functionality.

5

Get Pairing Code

Complete setup on child's device to get the 6-digit pairing code.

6

Install Parent App

Install Parent App on your phone, create account, and enter the pairing code.
